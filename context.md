Proyecto Tracker:
Queremos crear un sistema que permita llevar registro de actividades del dia a dia y compartirlo y compararlo con amigos. El usuario debe poder registrarse y crear grupos en los que puede invitar a otros usuarios. Dentro del grupo se pueden crear actividades de distinto tipo y los usuarios pueden cargar cuando la realizan. El usuario debe poder compararse con los otros usuario del grupo.

Funcionalidades minimas:

Login: el usuario debe poder registrarse y loguearse.
Groups: el usuario debe poder crear grupos e invitar a otros usuarios.
Activities: el usuario debe poder crear actividades dentro del grupo.
Records: el usuario debe poder cargar cuando realiza una actividad.
Comparisons: Se debe poder ver el progreso de todos los participantes del grupo en cada actividad. 
Dashboard: El proyecto acutal cuenta con un dashboard para el que el admin vea la db. movelo a "/panel" y que sea solo accesible para administradores. Cambia el index por uno especifico para nuestro proyecto.

Usa como base el proyecto existente y adaptalo al nuevo dominio de negocio. Mantené los estándares de calidad de código, los patrones de diseño y la estructura del proyecto.
